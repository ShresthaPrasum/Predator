# Time Bomb

## A​ 2d local multiplayer game where you pass the bomb and dodge another player from passing the bomb to you by running and jumping. You play as a ragdoll so it's quite difficult and fun to move the player!

## Just pass the bomb and don't let the another player to pass the bomb to you ! There are some easter eggs so if you find one tell it on the comment!

## There are multiple boosters such as speed, jump and shield which can help you to win the game.

# Controls!
- RED: WAD
- BLUE: Arror Key
